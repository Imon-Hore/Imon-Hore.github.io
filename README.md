# Imon Hore — Personal Portfolio

🌐 **Live Site:** https://imon-hore.github.io

---

## About

Personal portfolio website built with pure HTML, CSS and JavaScript.
No frameworks, no dependencies — fully self-contained single file.

---

## Features

- Downloadable Resume (PDF embedded)
- Working Contact Form (Web3Forms)
- Animated particle background
- Fully responsive (mobile + desktop)
- Smooth scroll navigation
- Intersection Observer animations

---

## Sections

- Hero — Introduction & quick stats
- About — Skills & background
- Projects — Featured work
- Experience — Timeline
- Awards — Recognitions
- Contact — Form + social links

---

## Tech Used

- HTML5, CSS3, Vanilla JavaScript
- Web3Forms (contact form backend)
- Google Fonts (Syne, Inter, Space Mono)
